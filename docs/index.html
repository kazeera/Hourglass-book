User’s Guide for Hourglass

User’s Guide for Hourglass

Written by Kazeera Aliar

last revised Aug 26, 2022

1 Preface

This is the official guide for Hourglass, the tool to systematically interrogate multiparametric datasets in a semi-automated manner. It allows users to filter, subset datasets and perform analysis under identical settings at the global, regional and patient-paired levels to identify communalities and differences.

Figure 1.1: The Hourglass logo.

Hourglass exists as an open-source desktop application for Windows/MacOS and a stand-alone R package. Please refer to our link paper for more information.

Figure 1.2: The Hourglass concept: input data, computational processing, output graphs.
